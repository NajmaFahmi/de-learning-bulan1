--> print working directory 

(venv_de) najma@192 week3 % pwd

## touch
--> create new file

(venv_de) najma@192 week3 % touch hanyatest.txt
(venv_de) najma@192 week3 % touch ./linux/inicontoh.txt


## mkdir 
--> create new directory

(venv_de) najma@192 week3 % mkdir foldertest


## rm 
--> delete file 

(venv_de) najma@192 week3 % rm hanyatest.txt


## rm -rf
--> delete directory with its files

(venv_de) najma@192 week3 % rm -rf foldertest
